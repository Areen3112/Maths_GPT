# 🧮 MathsGPT

An AI-powered mathematical assistant built with **Streamlit** and **Gemma LLM** that solves mathematical problems and provides step-by-step explanations in real time.

---

## 🚀 Features

* ✍️ Accepts natural language math questions
* 🤖 Powered by **Gemma model** for intelligent reasoning
* 📊 Provides step-by-step solutions
* ⚡ Fast and interactive Streamlit UI
* 🎯 Useful for students, exam preparation, and learning concepts

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Gemma LLM
* LangChain (if used internally)
* Google/LLM API (depending on setup)
* dotenv (for environment variables)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yAreen3112/Maths_GPT.git
cd MathsGPT
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the App

Start the Streamlit application:

```bash
streamlit run app.py
```

---

## 🧠 How It Works

1. User enters a mathematical question in natural language.
2. The input is processed and sent to the Gemma LLM.
3. The model interprets the problem and generates a step-by-step solution.
4. The result is displayed in a clean Streamlit interface.

## 💡 Example Queries

* Solve: “What is the derivative of x² + 3x + 5?”
* “Integrate x² from 0 to 5”
* “Solve quadratic equation x² + 5x + 6 = 0”
* “What is 15% of 240?”
